# Auth System

Sistema completo de autenticação, desenvolvido com **NestJS** no backend e **Next.js** no frontend, seguindo boas práticas de arquitetura, segurança e versionamento.

---

## Tecnologias Utilizadas

### Backend
- **Node.js**
- **NestJS**
- **JWT (JSON Web Token)**
- **bcrypt**
- **TypeScript**
- **PostgreSQL** (configuração preparada)

### Frontend
- **Next.js (App Router)**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Context API**

### Ferramentas
- **Git & GitHub**
- **Semantic Versioning (SemVer)**

---

## Funcionalidades

- Cadastro de usuários
- Login com autenticação JWT
- Hash seguro de senhas
- Contexto global de autenticação no frontend
- Logout
- Rotas protegidas (base)
- Interface moderna, responsiva e mobile-first
- Estrutura pronta para:
  - Refresh Token
  - Middleware de proteção
  - Controle de permissões (roles)
